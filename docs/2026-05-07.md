HTML의 기본 구조

                <!DOCTYPE HTML> : HTML 버전에 해당하는 문서 읽기(5버전)
                <html> : 문서의 시작
                  <head>
                    설정 파일 
                    <link> : CSS파일 읽기 => 공통으로 사용되는 CSS가 있는 경우 (파일)
		                <title> : 브라우저 제목 설정
				            <script> : 자바스크립트 
            				<style> : 인라인 CSS => 파일 한개에만 제어
            				<meta> : 한글 변환 => UTF-8, EUC-KR
            			  => 브라우저에 출력하지 않는다
                  </head>
                  <body>
                    => 브라우저에 출력이 가능한 태그
                  </body>
                </html> : 문서의 종료

DOCTYPE : HTML버전 선언
html : 문서 루트
head : 문서 정보
body : 실제 컨텐츠

<body> 안에 출력 => 브라우저에 출력
1) 텍스트 관련 
  
    제목 : h1~h6 => block => 다음 줄에 출력, 굵은 글씨체
        				 ------
    문자제어 : <b> 굵은글씨, ***<strong> 강조체, ***<font> 색상 글꼴 , ***<sup> 위첨자
        			
    문단제어 : ***<br> 다음줄 출력, ***<p> 단락 나누기, **<hr> 수평선, ***<pre> 있는 그대로
        			
2) 멀티미디어

       이미지 : <img src="이미지파일" title="소개">  *(원래 alt를 사용했으나 5버전에서는 title로 변경)
       동영상 : <embed src="동영상" width="" height="">
       다른 파일 : <iframe src="다른 파일명" width="" height="">

문자 제어 태그 
    
     <small></small> : 작은 글자 출력
     <address></address> : 주소 출력 => 이태릭체
     <sup></sup> 위첨자 : 새글 등록
     <b></b> : bold => 두꺼운 글자체
     <sub></sub> : 아래첨자 
     <em></em> : 강조체 => <strong></strong>
     <i></i> : 이태릭
     <mark></mark> : 형광색 출력
     <marquee></marquee> : 전광판 효과
