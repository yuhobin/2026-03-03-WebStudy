### 🏷️ Topic
HTML의 table 태그와 input태그 

### 📝 What I Learned
구조적인 표(table)을 만들고 input 태그를 통해 다양한 기능들을 실행 시켜보았다.

table : 

  <table> : <table> 전체 표를 감싸는 베이스 보드 
    <thead>
      <tr> : <tr> 데이터를 담을 가로 줄 table row
        <th>구분</th>  : <th> 굵은 글씨와 중앙 정렬이 기본인 제목 셀  table head
        <th>항목</th>
      </tr>
      <tr>
        <th>(대분류)</th>
        <th>(소분류)</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td></td> : <td> 실제 데이터가 들어가는 일반 셀 table data
        <td></td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <td></td>
        <td></td>
      </tr>
    </tfoot>
  </table>

<colspan> : colspan='숫자' 가로 열 합치기
<rowspan> : rowspan='숫자' 세로 열 합치기
