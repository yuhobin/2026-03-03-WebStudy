CSS 선택자
```html
1. 기본 선택자
태그 선택자: 태그명 { }
특정 태그 전체에 스타일을 적용 (h1)

id 선택자: #id명 { }
한 페이지 내에서 중복 없이 단 하나만 지정할 때 사용 (#red)

class 선택자: .class명 { }
중복 사용이 가능하며, 여러 태그를 그룹으로 묶어 공통 스타일을 줄 때 사용 (.blue)
```
```html
2. 관계 선택자
태그 사이의 부모, 자식, 형제 관계를 이용하여 특정 요소를 선택

후손 선택자 (태그 태그): 공백으로 구분하며, 부모 태그 안에 포함된 모든 해당 태그(자식, 손자 등)를 선택

자식 선택자 (태그 > 태그): 부모 태그 바로 아래에 있는 자식 태그만 선택

인접 형제 선택자 (태그 + 태그): 특정 태그 바로 다음에 오는 첫 번째 형제 태그 하나만 선택

일반 형제 선택자 (태그 ~ 태그): 특정 태그 다음에 오는 모든 형제 태그를 선택
```
```html
3. 속성 선택자
태그 안에 적힌 특정 속성 값에 따라 선택

[href]: href 속성이 있는 모든 요소

[type="text"]: type이 정확히 "text"인 요소

[class^="icon"]: 클래스명이 "icon"으로 시작하는 요소

[class$="icon"]: 클래스명이 "icon"으로 끝나는 요소
```
