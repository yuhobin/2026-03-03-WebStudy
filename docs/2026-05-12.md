### 📝 What I Learned
오늘은 JSP의 동작 원리를 파악하고, DB 데이터를 웹 화면에 동적으로 출력하는 실습과 CSS 선택자를 활용해 이를 꾸미기 위한 CSS 심화 과정을 학습했습니다. 


#### 1. JSP (Java Server Page) 핵심
```html
JSP는 HTML 코드 안에 자바를 넣어 동적 페이지를 만드는 서버 실행 파일이다
```
JSP의 특징:
```html
 정적인 HTML과 달리 데이터 변경이 가능하며,
 실행 시 Tomcat 서버에 의해 자바 파일 Servlet으로 변환되어 컴파일된 후 브라우저에 전달된다
```

주요 문법:
```html
지시자 (<%@ %>): 페이지 정보 설정 및 다른 파일(DAO 등)을 불러올 때 사용

스크립틀릿 (<% %>): 일반적인 자바 코드(변수 선언, 제어문)를 작성

표현식 (<%= %>): 자바 변수나 연산 결과를 화면에 바로 출력
```
작업 흐름
```html
자바에서 데이터를 수집(DAO) ->  HTML로 데이터를 출력 ->  CSS로 화면 UI를 완성
```
#### 2. CSS 선택자 및 속성 
브라우저의 요소를 정밀하게 제어하고 박스 모델을 통해 레이아웃을 잡는 방법을 익혔다

```html
기본 선택자: 태그, .클래스, #아이디 (우선순위: 아이디 > 클래스 > 태그)

복합 선택자 (관계):

하위(공백): 모든 후손 요소 선택

자식(>): 직계 자식만 선택

인접 형제(+): 바로 뒤의 형제 하나만 선택

일반 형제(~): 뒤에 오는 모든 형제 선택

속성 선택자 ([]): [type="text"]처럼 특정 속성값을 가진 요소 지목
```

가상 클래스 (:):
```html
상태: :hover(마우스 오버), :checked(체크 시)

구조: :first-child, :last-child, :nth-child(n) (순서 제어)
```

박스 모델 (Box-Model):
```html
Margin: 외부 간격 및 중앙 정렬(0px auto)

Padding: 내부 간격 및 글자 여백 조절

Border: 테두리 스타일 및 모서리 곡률 radius 처리
```

### 💡 Note
JSP의 개념을 배우고 이해하기 위해서 JSP파일에서 Java문서를 작성하는 방법을 익혔는데,
확실히 Java를 활용해 쿼리문을 날려 DB를 연동하는 과정을 거치니 더 완성된 웹 브라우저가 만들어졌다
CSS는 선택자를 활용할 수 있어야 크롤링하는데 문제없이 진행 할 수 있다.
