#### 1. CSS 기본 구조

CSS는 **선택자**와 **속성**으로 구성된다.

```css
선택자 {
  속성: 값;
}
```

| 구분 | 역할 | 예시 |
|------|------|------|
| **선택자 (Selector)** | 스타일을 적용할 HTML 요소를 지정 | `h1`, `.class`, `#id` |
| **속성 (Property)** | 선택된 요소에 어떤 스타일을 줄지 정의 | `color`, `width`, `display` |



---

#### 2. 선택자 (Selector)

선택자는 스타일을 적용할 HTML 요소를 선택하는 방법이다.

| 종류 | 문법 | 설명 |
|------|------|------|
| 태그 선택자 | `h1 { }` | 특정 HTML 태그 전체 선택 |
| 클래스 선택자 | `.className { }` | class 속성이 일치하는 요소 선택 |
| ID 선택자 | `#idName { }` | id 속성이 일치하는 요소 선택 |
| 전체 선택자 | `* { }` | 모든 요소 선택 |
| 자식 선택자 | `div > p { }` | div의 직계 자식 p 선택 |
| 후손 선택자 | `div p { }` | div 안의 모든 p 선택 |
| 인접 선택자 | `h1 + p { }` | h1 바로 다음 형제 p 선택 |
| 속성 선택자 | `input[type="text"] { }` | 특정 HTML 속성값을 가진 요소 선택 |
| 가상 클래스 | `a:hover { }` | 특정 상태의 요소 선택 |

```css
/* 예시 */
h1 { color: red; }            /* 태그 선택자 */
.title { font-size: 24px; }   /* 클래스 선택자 */
#header { background: navy; } /* ID 선택자 */
```

---

#### 3. 속성 (Property)

속성은 선택된 요소에 적용할 스타일의 종류이다.

---

#### 3-1. 박스 속성 (Box)

HTML의 모든 요소는 **박스(Box)** 구조로 이루어져 있다.

```
┌──────────────────────────────────┐
│             margin               │
│   ┌──────────────────────────┐   │
│   │         border           │   │
│   │   ┌──────────────────┐   │   │
│   │   │     padding      │   │   │
│   │   │  ┌────────────┐  │   │   │
│   │   │  │  content   │  │   │   │
│   │   │  │ width×height│  │   │   │
│   │   │  └────────────┘  │   │   │
│   │   └──────────────────┘   │   │
│   └──────────────────────────┘   │
└──────────────────────────────────┘
```

| 속성 | 설명 | 예시 |
|------|------|------|
| `width` | 요소의 너비 | `width: 300px` |
| `height` | 요소의 높이 | `height: 200px` |
| `margin` | 테두리 바깥 여백 | `margin: 10px 20px` |
| `padding` | 테두리 안쪽 여백 | `padding: 10px` |
| `border` | 테두리 스타일 | `border: 1px solid black` |
| `box-sizing` | 박스 크기 기준 설정 | `box-sizing: border-box` |

```css
/* margin / padding 단축 표기 */
margin: 10px;                  /* 상하좌우 모두 10px */
margin: 10px 20px;             /* 상하 10px, 좌우 20px */
margin: 10px 20px 15px 5px;    /* 상 우 하 좌 (시계 방향) */

/* border 단축 표기 */
border: 두께 스타일 색상;
border: 2px solid #333;

/* box-sizing */
box-sizing: content-box;  /* 기본값: width가 content 영역만 포함 */
box-sizing: border-box;   /* width가 padding + border 포함 (권장) */
```

#### 3-2. 가시 속성 (Visibility)

요소를 **화면에 어떻게 표시할지**, 혹은 **보이게/숨길지** 제어한다.

| 속성 | 설명 | 예시 |
|------|------|------|
| `display` | 요소의 배치 방식 설정 | `display: block` |
| `visibility` | 요소의 시각적 표시 여부 | `visibility: hidden` |
| `opacity` | 요소의 투명도 (0.0 ~ 1.0) | `opacity: 0.5` |

##### display 주요 값

| 값 | 설명 |
|----|------|
| `block` | 한 줄 전체 차지, width/height 지정 가능 |
| `inline` | 내용 너비만큼만 차지, width/height 지정 불가 |
| `inline-block` | inline처럼 나란히 배치 + width/height 지정 가능 |
| `none` | 요소를 완전히 숨김 (공간도 차지하지 않음) |

##### visibility vs display:none 차이

| 속성 | 화면에 보임 | 공간 차지 |
|------|------------|----------|
| `visibility: hidden` | ❌ | ✅ |
| `display: none` | ❌ | ❌ |

```css
/* 예시 */
div { display: none; }        /* 공간도 사라짐 */
div { visibility: hidden; }   /* 공간은 유지, 보이지만 않음 */
div { opacity: 0; }           /* 완전 투명, 공간 유지, 클릭도 됨 */
```

#### 3-3. 배경 속성 (Background)

요소의 **배경색, 배경 이미지** 등을 설정한다.

| 속성 | 설명 | 예시 |
|------|------|------|
| `background-color` | 배경 색상 | `background-color: #f0f0f0` |
| `background-image` | 배경 이미지 | `background-image: url('img.png')` |
| `background-repeat` | 배경 이미지 반복 방식 | `background-repeat: no-repeat` |
| `background-position` | 배경 이미지 위치 | `background-position: center` |
| `background-size` | 배경 이미지 크기 | `background-size: cover` |
| `background-attachment` | 배경 이미지 스크롤 여부 | `background-attachment: fixed` |

##### background-repeat 값

| 값 | 설명 |
|----|------|
| `repeat` | 기본값, 가로·세로 반복 |
| `repeat-x` | 가로 방향만 반복 |
| `repeat-y` | 세로 방향만 반복 |
| `no-repeat` | 반복하지 않음 |

##### background-size 값

| 값 | 설명 |
|----|------|
| `cover` | 요소를 완전히 덮도록 확대 (잘릴 수 있음) |
| `contain` | 이미지 전체가 보이도록 맞춤 |
| `100px 200px` | 직접 크기 지정 |

```css
/* 단축 속성 (background shorthand) */
background: 색상 이미지 반복 위치 / 크기;

/* 예시 */
background: #fff url('bg.png') no-repeat center / cover;

/* background-attachment 예시 */
background-attachment: scroll;  /* 기본값: 스크롤 시 배경도 같이 이동 */
background-attachment: fixed;   /* 스크롤해도 배경은 고정 (패럴랙스 효과) */
```
