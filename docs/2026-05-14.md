# 📅 2026-05-14 CSS_Visibility/Font
### 🏷️ Topic
CSS의 가시속성과 글자속성

### 📝 What I Learned
### 1. 가시 속성 (Visibility)

요소를 **화면에 어떻게 표시할지**, 혹은 **보이게/숨길지** 제어한다.

| 속성 | 설명 | 예시 |
|------|------|------|
| `display` | 요소의 배치 방식 설정 | `display: block` |
| `visibility` | 요소의 시각적 표시 여부 | `visibility: hidden` |
| `opacity` | 요소의 투명도 (0.0 ~ 1.0) | `opacity: 0.5` |

#### display 주요 값

| 값 | 설명 | 기본 적용 태그 |
|----|------|--------------|
| `block` | 한 줄 전체 차지, width/height 지정 가능 | `div`, `p`, `h1` |
| `inline` | 내용 너비만큼만 차지, width/height 지정 불가 | `span`, `a` |
| `inline-block` | inline처럼 나란히 배치 + width/height 지정 가능 | - |
| `none` | 요소를 완전히 숨김 (공간도 차지하지 않음) | - |
| `flex` | 자식 요소를 유연하게 배치 | - |

#### visibility / opacity / display:none 비교

> 셋 다 요소를 숨기지만 동작 방식이 다르다.

| 속성 | 화면에 보임 | 공간 차지 | 클릭 이벤트 |
|------|------------|----------|------------|
| `visibility: hidden` | ❌ | ✅ | ❌ |
| `opacity: 0` | ❌ | ✅ | ✅ |
| `display: none` | ❌ | ❌ | ❌ |

```css
div { display: none; }        /* 공간도 사라짐 */
div { visibility: hidden; }   /* 공간은 유지, 보이지 않음 */
div { opacity: 0; }           /* 완전 투명, 공간 유지, 클릭 가능 */
div { opacity: 0.5; }         /* 반투명 */
```

### 2. 글자 속성 (Font / Text)

텍스트의 **크기, 굵기, 정렬, 색상** 등을 설정한다.

---

### 2-1. 폰트 속성 (font-)

| 속성 | 설명 | 예시 |
|------|------|------|
| `font-size` | 글자 크기 | `font-size: 16px` |
| `font-weight` | 글자 굵기 | `font-weight: bold` |
| `font-style` | 글자 기울기 | `font-style: italic` |
| `font-family` | 글꼴 설정 | `font-family: Arial, sans-serif` |

#### font-size 단위

| 단위 | 설명 |
|------|------|
| `px` | 고정 크기 (가장 많이 사용) |
| `em` | 부모 요소 기준 상대 크기 |
| `rem` | html 루트 기준 상대 크기 |
| `%` | 부모 요소 기준 퍼센트 |

#### font-weight 주요 값

| 값 | 설명 |
|----|------|
| `normal` | 기본값 (400과 동일) |
| `bold` | 굵게 (700과 동일) |
| `100` ~ `900` | 숫자로 직접 지정 (100 단위) |

```css
p {
  font-size: 16px;
  font-weight: bold;       /* 또는 font-weight: 700 */
  font-style: italic;
  font-family: "Noto Sans KR", sans-serif; /* 앞에서부터 순서대로 적용, 없으면 다음 폰트 */
}

/* font 단축 속성 */
/* font: 기울기 굵기 크기/줄간격 폰트명 */
font: italic bold 16px/1.5 Arial, sans-serif;
```

---

### 2-2. 텍스트 속성 (text-)

| 속성 | 설명 | 예시 |
|------|------|------|
| `color` | 글자 색상 | `color: #333` |
| `text-align` | 텍스트 정렬 | `text-align: center` |
| `text-decoration` | 텍스트 꾸밈 (밑줄 등) | `text-decoration: none` |
| `text-indent` | 첫 줄 들여쓰기 | `text-indent: 20px` |
| `line-height` | 줄 간격 | `line-height: 1.6` |
| `letter-spacing` | 글자 간격 | `letter-spacing: 2px` |
| `word-spacing` | 단어 간격 | `word-spacing: 4px` |
| `white-space` | 공백 처리 방식 | `white-space: nowrap` |

#### text-align 값

| 값 | 설명 |
|----|------|
| `left` | 왼쪽 정렬 (기본값) |
| `right` | 오른쪽 정렬 |
| `center` | 가운데 정렬 |
| `justify` | 양쪽 정렬 |

#### text-decoration 값

| 값 | 설명 |
|----|------|
| `none` | 꾸밈 없음 (a태그 밑줄 제거 시 자주 사용) |
| `underline` | 밑줄 |
| `overline` | 윗줄 |
| `line-through` | 취소선 |

```css
p {
  color: #333;
  text-align: center;
  line-height: 1.6;      /* 단위 없이 배수로 쓰는 것을 권장 */
  letter-spacing: 1px;
}

/* a태그 밑줄 제거 시 자주 사용 */
a {
  text-decoration: none;
}

/* 텍스트 말줄임표 처리 (세 가지 세트로 사용) */
p {
  white-space: nowrap;           /* 줄바꿈 없이 한 줄로 */
  overflow: hidden;              /* 넘치는 내용 숨김 */
  text-overflow: ellipsis;       /* 넘치는 텍스트를 ...으로 표시 */
}
```

### 💡 Note
가시 속성을 배우면서, 화면에 보이는 결과는 비슷해 보여도 실제로는 완전히
제거한 것인지, 숨긴 것인지, 투명하게 처리한 것인지에 따라 표현 방식이
전혀 다르다는 점이 흥미로웠다.

표현 방식에 따라 공간을 차지하는 정도도 다르고, 클릭 같은 동작의 발생
여부도 달라진다는 것을 알게 됐다.

같은 결과처럼 보이는 화면 뒤에 이렇게 다양한 방식이 존재하고, 어떤 방식을
선택할지는 개발자의 판단에 달려 있다는 점에서 코드에도 사람마다의 관점과
스타일이 드러날 수 있다는 게 새롭게 느껴졌다.

```css
URL 작성시 주의사항
<a href="list.jsp?page=<%=curpage>1?curpage-1:curpage%>&type=<%=type %>" class="btn btn-sm btn-warning">이전</a>
여기서 &type= 이부분 &type = 으로 공백을 주지 않도록 주의한다
(공백이 생기면 url주소에서 공백을 인식하지 못할때가 있다.)
```
